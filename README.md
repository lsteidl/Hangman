# Hangman
Simple hangman game. 
Randomly selects word from wordlist file.
Displays incorrect guesses to user.
Prevents user from making same guess twice.

4 Difficulty levels...
1) Easy - Allows 10 wrong guesses
2) Medium - Allows 5 wrong guesses
3) Hard - Allows 3 wrong guesses
4) Classic Hangman - Allows 5 wrong guesses and uses hangman graphic
